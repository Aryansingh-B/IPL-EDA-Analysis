# IPL Data Analysis (2008–2023)

Exploratory Data Analysis on 10+ years of IPL data covering 950+ matches and 180,000+ ball-by-ball deliveries.

## Tech Stack
Python · pandas · numpy · matplotlib · seaborn

## Key Insights
- Toss win leads to match win ~52% of the time — marginal edge
- Mumbai Indians dominate with most wins across all seasons
- Virat Kohli is the all-time leading run scorer
- Lasith Malinga leads wicket takers
- Home advantage is strongest at Wankhede and Chepauk

## Dataset
[IPL Complete Dataset – Kaggle](https://www.kaggle.com/datasets/patrickb1912/ipl-complete-dataset-20082020)
